# Дополнительные WS-опкоды — WebApp, Complaints, Folders, Polls, Audio

## WebApp

| Опкод | Enum # | WS code | Что |
|---|---|---|---|
| `WEB_APP_INIT_DATA` | 139 | 160 | **инициализация мини-апки** — передача данных при старте |

`WEB_APP_INIT_DATA` — при запуске мини-апки клиент отправляет на сервер инициализационные данные. Это включает идентификатор мини-апки, параметры запуска, и возможно токен пользователя. Сервер получает факт запуска каждой мини-апки.

## Жалобы

| Опкод | Enum # | WS code | Что |
|---|---|---|---|
| `COMPLAIN` | 140 | 161 | отправить жалобу |
| `COMPLAIN_REASONS_GET` | 141 | 162 | получить список причин жалоб |

Подтверждение [[63-complaints-server-controlled]]: список причин жалоб получается с сервера через WS.

## Папки

|---|---|---|---|
| `FOLDERS_GET` | 142 | 272 | получить папки |
| `FOLDERS_GET_BY_ID` | 143 | 273 | получить папку по ID |
| `FOLDERS_UPDATE` | 144 | 274 | обновить папку |
| `FOLDERS_REORDER` | 145 | 275 | изменить порядок папок |
| `FOLDERS_DELETE` | 146 | 276 | удалить папку |

## Опросы

| Опкод | Enum # | WS code | Что |
|---|---|---|---|
| `SEND_VOTE` | 154 | 304 | **отправить голос** |
| `VOTERS_LIST_BY_ANSWER` | 155 | 305 | список проголосовавших по ответу |

`VOTERS_LIST_BY_ANSWER` — сервер может вернуть список пользователей, проголосовавших за конкретный вариант. Это означает, что голосование в MAX **не анонимное** — сервер знает, кто как проголосовал.

## Аудио

| Опкод | Enum # | WS code | Что |
|---|---|---|---|
| `AUDIO_PLAY` | 152 | 301 | **воспроизведение аудио** |

`AUDIO_PLAY` — клиент уведомляет сервер о воспроизведении аудиосообщения. Сервер знает, когда пользователь прослушивает голосовые сообщения.

## Организации

| Опкод | Enum # | WS code | Что |
|---|---|---|---|
| `ORG_INFO` | 159 | 256 | информация об организации |

## Что важно

1. **`WEB_APP_INIT_DATA(160)`** — сервер знает о каждом запуске мини-апки.
2. **`VOTERS_LIST_BY_ANSWER(305)`** — голосование не анонимное.
3. **`AUDIO_PLAY(301)`** — сервер знает, когда пользователь прослушивает голосовые.

## Сводка

`WEB_APP_INIT_DATA` — факт запуска каждой мини-апки. `VOTERS_LIST_BY_ANSWER` — неанонимное голосование. `AUDIO_PLAY` — сервер знает о прослушивании голосовых. 5 опкодов для папок.
