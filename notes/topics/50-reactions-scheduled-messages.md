# Реакции и отложенные сообщения — server-gated фичи

## Реакции

### WS-опкоды реакций

| Опкод | Enum # | WS code | Что |
|---|---|---|---|
| `REACTIONS_SETTINGS_GET_BY_CHAT_ID` | 59 | 258 | получить настройки реакций для чата |
| `CHAT_REACTIONS_SETTINGS_SET` | 60 | 257 | **установить настройки реакций для чата** |
| `MSG_REACTION` | 66 | 178 | поставить реакцию |
| `MSG_CANCEL_REACTION` | 67 | 179 | убрать реакцию |
| `MSG_GET_REACTIONS` | 68 | 180 | получить список реакций на сообщение |
| `MSG_GET_DETAILED_REACTIONS` | 69 | 181 | детальный список (кто поставил) |
| `NOTIF_MSG_REACTIONS_CHANGED` | 121 | 155 | push-уведомление об изменении реакций |

### PmsKey реакций

| Ключ | # | Что |
|---|---|---|
| `reactions-max` | 54 | максимальное количество типов реакций |
| `reactions-settings-enabled` | 215 | включить настройки реакций per-chat |
| `default-reactions-settings` | 216 | настройки реакций по умолчанию (JSON) |
| `reactions-sync-time` | (в rtd) | интервал синхронизации реакций |
| `msg-get-reactions-page-size` | (в rtd) | размер страницы при получении реакций |

### Что важно

`CHAT_REACTIONS_SETTINGS_SET(257)` — **администратор чата может через WS задать, какие реакции разрешены в конкретном чате**. Это server-side per-chat политика реакций. `MSG_GET_DETAILED_REACTIONS(181)` — сервер знает, кто именно поставил каждую реакцию (не анонимно).

## Отложенные сообщения

### PmsKey

| Ключ | # | Что |
|---|---|---|
| `scheduled-messages-enabled` | 48 | отложенные сообщения в чатах |
| `scheduled-posts-enabled` | 49 | отложенные посты в каналах |
| `scheduled-faves-enabled` | 50 | отложенные сообщения в «Избранном» |

Все три — server-gated bool. По умолчанию неизвестно (нет явного default в коде, в отличие от polls). Три отдельных ключа означают, что сервер может разрешить отложенные сообщения в каналах, но запретить в личных чатах.

### Что важно

Отложенные сообщения хранятся на сервере (не локально). Это означает, что сервер знает содержимое запланированных, но ещё не отправленных сообщений. В случае государственного мессенджера — это потенциально доступно для просмотра до отправки.

## Сводка

7 WS-опкодов для реакций (включая per-chat настройки и детальный список «кто поставил»). 3 PmsKey для отложенных сообщений с гранулярным контролем по типу чата. Отложенные сообщения хранятся на сервере до момента отправки.
